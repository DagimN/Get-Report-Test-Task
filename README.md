# Mobile-Only Transaction Tracker

React + TypeScript + Tailwind CSS implementation of a mobile-only transaction tracker with:

- Transaction list and transaction detail screens
- Seasonal daily points formula
- Data loaded from `data.json` via a custom hook (`useTransactions`)
- FontAwesome-based transaction icons
